List

1.What is list?

Ans:-List is a collection or set of many items,each specified in a separate line

Types of html list with definition.

1.What is ordered list?

Ans:-An ordered list is a list where all items are numbered. In this list the numbered can be replaced with roman numbers or the letters of english alphabet.It represent item in the list in specific order.

In creating ordered list ,< ol > and < li >are used.

2.What is unordered list?

Ans:-An unordered list is bulleted list.Unordered list is used when items in the list are not required to be in specific order.

In creating unordered list , < ul > and < li >are used.

3.What is description or definition list?

Ans:-A description or definition list is a list with a definition or description of each item.

In creating description list or definition list , < dl > , < dd > and < dt > are used .

4.What is nested list?

Ans:-A list enclosed within another list is called nested list.

In creating nested list, < ul > , < ol > and < li > are used.
